problemas encontrados e como eles foram resolvidos.

Embora esses problemas existam, acredito que o projeto está bem estruturado e organizado.

## Bootstrap

Ja na estrura no root existe uma alteração que é importante. Diferente de outros aplicativos react com vite
onde arquivo `index.html` define o entry point da aplicação com o script:

```html
<script type="module" src="/src/main.tsx"></script>
```

Nesse projeto as coisas acontecem diferente, ao invés de definir um `script` com um `src`
estático no arquivo `index.html`, esse `script` é inserido pelo `vite` na hora do build. Isso é
feito via plugin do `vite` que está definido no próprio `vite.config.ts`, o plugin se chama `entry-point-by-mode`.

Ao decorrer da documentação vou explicar um pouco mais sobre o `entry-point-by-mode` e como ele funciona.
