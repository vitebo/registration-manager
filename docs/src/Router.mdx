# `~/router/`

Esta pasta não teve atualizações recentes, mas tem a responsabilidade de definir as rotas da aplicação e qual componente
renderizar de acordo com cada rota. Inclusive, acredito que renomear a rota de `/new-user` para `/new-registration`
faria bastante sentido. Usar nomes em `pt-BR` na URL também parece ser bem-vindo, já que todo o texto da aplicação está em português.

Não explorei em detalhes como funciona o lazy loading de páginas em uma SPA React, mas isso também seria implementado
nesta camada.
