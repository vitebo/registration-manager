# `~/drivers/`

Nesta pasta, temos as implementações de infraestrutura da aplicação. Aqui estão as implementações de `http-client` e
`storage`. Essas implementações são utilizadas para abstrair a comunicação com a API e, por exemplo, com o
`localStorage`. Dessa forma, é fácil criar um *adapter* para um *storage*, utilizando uma biblioteca (ou não), e é
simples trocar a implementação.

A ideia principal é evitar o acoplamento com um *driver* específico ou com uma biblioteca que utiliza esse *driver*.
