# `~/validations/`

Diferente do que pode parecer `validations` não cuida das regras para validar um dado, isso é responsabilidade
da entidade e seus `value objects`. Validations apenas tenta criar o `value object` a partir da entrada do usuário.
Os `value objects` retornam uma `Exception` diferente para cada tipo de erro que pode acontecer,
validations apenas faz uma tradução da `Ecxeption` do `value object` para uma mensagem amigável para o usuário que
vai ser exibida na tela.

Regras de internacionalização referente a mensagens de erro são tratadas nessa camada.
